Remote control demo

MORSimulator is a 3D robotic simulator running on a web browser. Click the following link to view the desktop environment. Check the info page for more details.

MORSimulator runs on a local physics engine which requires some processing capacity, thus the mobile experience can lead to some frustration. MORSimulator attempts to circumvent this lack of processing power by leaving the physics engine computation to a PC and only rendering the 3D world in the mobile devices.

This demo showcases the ability to interact with any desktop environment from a mobile device (only tested in Android devices). Get the connection code from desktop version of MORSimulator and connect from this mobile device. Once connected the mobile orientation will guide the vehicle.
